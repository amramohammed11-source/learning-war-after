أسئلة شائعة

Accordion مع Animation باستخدام jQuery (Slide).

اضغط زر فتح داخل جدول المواد، سيظهر الملف في نافذة منبثقة، ويمكنك أيضًا فتحه في تبويب جديد.
يجب تشغيل المشروع عبر VS Code باستخدام Go Live (عنوان 127.0.0.1) وليس عبر file://.
نعم، زر المرحلة يبدّل المواد والفيديوهات تلقائيًا حسب المرحلة المختارة.
بعض الفيديوهات تمنع التضمين (Embed). لذلك وضعنا خيار تشغيل داخل مودال وخيار فتح مباشر في YouTube.
